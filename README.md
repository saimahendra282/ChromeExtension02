## **Follow these steps:**

- Download this zip file from the releases section.<br>
  <img src="https://github.com/user-attachments/assets/ae086cf0-83c9-47a4-a600-3ba1884b9179" width="500"/>
  <br><img src="https://github.com/user-attachments/assets/e83a48a1-adf9-461a-ad93-291e720ffdd7" width="500"/>
- Extract the zip file.
- Next, you have to add your own Gemini API key.  
- For creating the API key, follow these steps (or refer online if you can't understand this):  
  1. First, go to [https://aistudio.google.com/](https://aistudio.google.com/)
  2. Sign in or create your account.
  3. Then you can find this **Get API** button on the top-right corner — click it.<br>
     <img src="https://github.com/user-attachments/assets/3492ff89-a121-4347-b248-b1a2af94231a" width="500"/>
  4. Click on **Create API Key**.<br>
     <img src="https://github.com/user-attachments/assets/fc3000b8-81b9-4c59-9597-bcad97f5c9b2" width="500"/>
  5. After creating, copy the key and come back to the repo folder.
  6. Create a `.env` file.
  7. Add the API key in the `.env` file like this:  
    ```
    GEMINI_API_KEY=
    ```

- Open **Chrome** and go to the **Extensions** page.<br>
  <img src="https://github.com/user-attachments/assets/9cf39eb2-cc61-4186-8df0-019ff426fd58" width="500"/>

- In the top-right corner, enable **Developer mode**.<br>
  <img src="https://github.com/user-attachments/assets/44f846d0-99df-4b9e-b30e-45e3a174afb0" width="500"/>

- Click the **Load unpacked** button and upload this repo folder.<br>
  <img src="https://github.com/user-attachments/assets/3caa3f87-0964-4bb4-9c3c-365843dd0195" width="500"/>
